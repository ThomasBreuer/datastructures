2 Installation

The release version and the development version of the package require slightly different processes for compilation of the kernel module. The package does not work without compiling this module

2.1 Release version

To install this package, extract the package's archive file into GAP's pkg folder.


      ./configure
      make

The datastructures is not loaded by GAP by default. To load the package run LoadPackage("datastructures"); at the GAP prompt.

2.2 Development version

To build the a development version of the package, first obtain the sources from github


      git clone https://github.com/gap-packages/datastructures

Install the required dependencies. If you are also using a development version of GAP, then you will have already installed them as part of its compilation process.

  • a C compiler, e.g. GCC or Clang

  • GNU Make

  • GNU Autoconf (we recommend 2.69 or later)

  • GNU Libtool

To build the kernel module then run the following commands.


      sh autogen.sh
      ./configure
      make

2.3 Building the documentation

To build the package documentation, run the following command in the package's directory


      gap makedog.g
